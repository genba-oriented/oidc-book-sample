= Clientがネイティブアプリの認可コードフローのプログラム

Clientが、Android のネイティブアプリの認可コードフローのサンプルプログラムです。

== 前提条件

[cols="1,2", options="header"]
|===
| 項目 | 要件

| Android Studio
| 最新版を推奨

| Keycloak
| 起動済みであること

| API サーバ
| 起動済みであること
|===

[NOTE]
====
* Android Studio のダウンロード → https://developer.android.com/studio
* Keycloak の起動方法 → link:../keycloak/README.adoc[keycloak/README.adoc]
* API サーバの起動方法 → link:../api/README.adoc[api/README.adoc]
====

== 実行手順

=== 1. プロジェクトを開く

Android Studio を起動し、本ディレクトリをプロジェクトとして開きます。

=== 2. アプリケーションの実行

Android エミュレータで実行します。
