= 「アプリケーション開発者のためのOpenID Connect 入門(仮)」のサンプルプログラム

本リポジトリは、書籍「アプリケーション開発者のためのOpenID Connect 入門(仮)」に掲載されているサンプルプログラムのリポジトリです。

各ディレクトリには、以下が格納されています。

|===
| ディレクトリ名 | 内容

|spa|ClientがSPAの認可コードフローのプログラム
|bff|ClientがBFFの認可コードフローのプログラム
|native|Clientがネイティブアプリの認可コードフローのプログラム
|batch|クライアントクレデンシャルフローのプログラム
|api|上記のプログラムが呼び出しているAPIサーバのプログラム
|keycloak|Keycloakを動かすための手順とデータ
|===

詳細は各ディレクトリ内の「README.adoc」をご参照ください。
