= API サーバのプログラム

各クライアントアプリケーション（SPA、BFF、ネイティブアプリ、バッチ）が呼び出す API サーバのサンプルプログラムです。

== 前提条件

[cols="1,2", options="header"]
|===
| 項目 | 要件

| Java
| 17 以上

| Keycloak
| 起動済みであること
|===

[NOTE]
====
Keycloak の起動方法は、書籍または link:../keycloak/README.adoc[keycloak/README.adoc] を参照してください。
====

== 実行手順

=== 1. ビルド

[source,bash]
----
mvnw clean package
----

=== 2. 起動

[source,bash]
----
java -jar target\oidc-book-sample-api-0.0.1-SNAPSHOT.jar
----
