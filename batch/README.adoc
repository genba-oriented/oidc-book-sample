= クライアントクレデンシャルフローのプログラム


クライアントクレデンシャルフローを実装したプログラムです。

== 前提条件

[cols="1,2", options="header"]
|===
| 項目 | 要件

| Java
| 17 以上

| Keycloak
| 起動済みであること

| API サーバ
| 起動済みであること
|===

[NOTE]
====
* Keycloak の起動方法 → link:../keycloak/README.adoc[keycloak/README.adoc]
* API サーバの起動方法 → link:../api/README.adoc[api/README.adoc]
====

== 実行手順

=== 1. ビルド

[source,bash]
----
mvnw clean package
----

=== 2. 実行

[source,bash]
----
java -jar target\oidc-book-sample-batch-0.0.1-SNAPSHOT.jar
----
