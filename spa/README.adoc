= ClientがSPAの認可コードフローのプログラム


ClientがSPA（Single Page Application）の認可コードフローのサンプルプログラムです。

== 前提条件

[cols="1,2", options="header"]
|===
| 項目 | 要件

| Node.js
| 最新の LTS 版を推奨

| Keycloak
| 起動済みであること

| API サーバ
| 起動済みであること
|===

[NOTE]
====
* Node.js のダウンロード → https://nodejs.org/ja/download
* Keycloak の起動方法 → link:../keycloak/README.adoc[keycloak/README.adoc]
* API サーバの起動方法 → link:../api/README.adoc[api/README.adoc]
====

== 実行手順

=== 1. ライブラリのインストール

[source,bash]
----
npm install
----

=== 2. アプリケーションの起動

[source,bash]
----
npm run dev
----

=== 3. ブラウザでアクセス

アプリケーションが起動したら、以下の URL にアクセスしてください。

----
http://localhost:3000
----
