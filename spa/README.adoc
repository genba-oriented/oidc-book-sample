= ClientがSPAの認可コードフローのプログラム

== 環境準備

* 最新のNode.jsをインストールしてください。
+
https://nodejs.org/ja/download

* Keycloakを起動してください。Keyclaokの起動方法は、書籍もしくは上位階層の「keycloak」ディレクトリをご参照ください。
* APIサーバを起動してください。APIサーバの起動方法は、上位階層の「api」ディレクトリをご参照ください。


== 実行方法
* ライブラリのインストール
+
....
npm install
....

* アプリケーションの実行
+
....
npm run dev
....

* ブラウザで以下にアクセス
+
....
http://localhost:3000
....
